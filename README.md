# dmhub-java-sdk

DM Hub OpenAPI 的 Java 封装

## 如何使用

### Gradle

```groovy
repositories {
    jcenter()
    maven { url 'https://jitpack.io' }
}
```

将 https://jitpack.io 加入仓库列表。

```groovy
dependencies {
    ...
    implementation 'com.github.flmn:trivial-java-utils:0.1.0'
    ...
}
```

将 com.github.flmn:trivial-java-utils 加入依赖。
